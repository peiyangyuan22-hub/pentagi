# V11 足球分析策略

**六步信号驱动框架 + 三方案串关构建**

## 文件

- `V11_analysis_strategy.py` — 完整实现（可独立运行测试）

## 分析流程

```mermaid
flowchart LR
    A[市场信号] --> B[基本面验证]
    B --> C[百家交叉验证]
    C --> D[赔率走势监控]
    D --> E[标签生成]
    E --> F[赔率门槛过滤]
    F --> G[输出]
```

## 标签系统

| 标签 | 含义 | 置信度 |
|------|------|--------|
| 🟢 绿标 | 多维度一致，信号强 | 高 |
| 🟡 黄标 | 信号中等，基本面支持 | 中 |
| ⚠️ 灰标 | 信号弱，回避 | 低 |
| 🔥 高赔候选 | 基本面与市场打架，极端排名 | 高风险高回报 |
| 🚫 不入 | 赔率范围外或不可靠 | 0 |

## 串关原则

1. **灰标不入串**
2. **绿标永远是锚**
3. **高赔来自"市场可能错了"的判断，不是来自数字大**
4. **赔率走势比绝对数值更有信息量**
5. **百家平均只做交叉验证，不做独立信号**
